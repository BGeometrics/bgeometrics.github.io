1. Aceptación de los Términos

Al acceder o utilizar el sitio web de BGeometrics (charts.bgeometrics.com) y cualquiera de sus servicios, incluidas las gráficas On-Chain gratuitas y la API de datos Bitcoin, aceptas quedar vinculado por estos Términos y Condiciones. Si no estás de acuerdo con alguno de ellos, te pedimos que dejes de utilizar nuestros servicios de inmediato.

Estos Términos son aplicables a todos los visitantes, usuarios registrados y consumidores de la API, tanto en el plan gratuito como en el de pago.

2. Descripción de los Servicios

BGeometrics ofrece dos tipos principales de servicios:

  • Gráficas On-Chain Gratuitas: Visualizaciones interactivas de métricas on-chain de Bitcoin como MVRV Z-Score, NUPL, SOPR, Precio Realizado, Hodl Waves, Hashrate y muchas otras. Todas las gráficas son de acceso libre sin necesidad de registro.
  • API de Datos Bitcoin: Una API REST que expone métricas on-chain y de mercado de Bitcoin en formato legible por máquinas. Está disponible de forma gratuita hasta los límites de uso descritos en la Sección 4, y mediante suscripción de pago para volúmenes superiores.

3. Gráficas On-Chain Gratuitas

Todas las gráficas y visualizaciones publicadas en BGeometrics se ofrecen de forma gratuita para uso personal, educativo y no comercial. No se requiere registro ni cuenta para visualizarlas.

Puedes compartir URLs de gráficas individuales o incrustarlas en contextos no comerciales siempre que:

  • Atribuyas claramente a BGeometrics como fuente.
  • No modifiques, extraigas de forma masiva ni redistribuyas las gráficas.
  • No las uses de manera que pueda inducir a error sobre su origen.

BGeometrics se reserva el derecho de modificar, actualizar o discontinuar cualquier gráfica en cualquier momento sin previo aviso. Los datos mostrados son meramente informativos y pueden contener retrasos o imprecisiones.

4. API de Bitcoin — Tier Gratuito y Límites de Uso

La API de Datos Bitcoin de BGeometrics está disponible para todos los usuarios con una clave API gratuita. Los siguientes límites de uso aplican al tier gratuito:

Tier Peticiones por Hora Peticiones por Día Acceso
Gratuito 8 15 Sin pago requerido
Pro / Pago Límites superiores Límites superiores Suscripción activa requerida

4.1 Condiciones del Tier Gratuito

  • El acceso gratuito a la API está sujeto a disponibilidad y puede ser limitado o pausado durante periodos de alta demanda.
  • Superar el límite horario o diario resultará en respuestas HTTP 429 (Demasiadas Solicitudes) hasta que el contador se restablezca.
  • El acceso gratuito está pensado para proyectos personales, investigación y evaluación de la API.
  • Está estrictamente prohibido usar scripts automatizados, bots o varias claves API para eludir los límites de uso.

4.2 Clave API

Para utilizar la API debes obtener una clave registrándote en bitcoin-data.com. Eres responsable de mantener tu clave API confidencial. Cualquier uso bajo tu clave es responsabilidad tuya. Si sospechas que tu clave ha sido comprometida, contáctanos de inmediato en info@bgeometrics.com.

4.3 Uso Aceptable de la API

La API puede utilizarse para:

  • Crear dashboards o herramientas personales o no comerciales.
  • Realizar investigaciones académicas o personales sobre Bitcoin.
  • Integrar datos en tus propias aplicaciones comerciales, sujeto al plan de suscripción correspondiente.

5. Suscripciones de Pago

Los usuarios que requieran un mayor volumen de peticiones a la API pueden suscribirse a un plan de pago. Las suscripciones se gestionan en bitcoin-data.com/bguser/pricing.

  • Las tarifas de suscripción se facturan según lo indicado en la página de precios en el momento de la compra.
  • Las suscripciones se renuevan automáticamente a menos que se cancelen antes de la fecha de renovación.
  • Los reembolsos no están garantizados y se evalúan caso por caso. Contáctanos en un plazo de 7 días desde el cargo si consideras que ha habido un error.
  • BGeometrics se reserva el derecho de modificar precios o características de los planes con previo aviso razonable.
  • El abuso de la API, incluso bajo un plan de pago, puede resultar en la suspensión sin reembolso.

6. Aviso Legal Financiero

El rendimiento pasado de Bitcoin o cualquier activo mostrado en esta plataforma no garantiza resultados futuros. Las métricas on-chain son herramientas analíticas y no predicen movimientos de precio. Realiza siempre tu propia investigación y consulta a un asesor financiero cualificado antes de tomar decisiones de inversión.

BGeometrics no se hace responsable de ninguna pérdida o daño económico resultante del uso de la información proporcionada en esta plataforma.
