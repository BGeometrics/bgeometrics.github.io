Services


At BGeometrics we specialize in Bitcoin on-chain data. We provide free interactive charts, a Bitcoin Data API, and custom development services for teams and projects that need reliable, up-to-date Bitcoin metrics. Everything we build is designed to be simple to use and easy to integrate.

API Plans & Pricing
Free Bitcoin On-Chain Charts
All Bitcoin on-chain charts available at BGeometrics are completely free, with no registration required. We cover the most relevant metrics used by analysts and investors worldwide:

• MVRV Z-Score, NUPL, SOPR, Realized Price, Hodl Waves, CDD, AVIV, Reserve Risk, Puell Multiple, Hashrate, Hashribbons, Power Law, and many more.
• Charts are interactive, built with Highcharts and Plotly, and updated daily.
• Available in light and dark mode, with zoom, pan, and export options.
• Embeddable via iframe in any website with a single line of code.
Bitcoin Data API — Free Tier
We offer a free Bitcoin Data API so you can access on-chain metrics programmatically in your own projects. No credit card required — just register and get your API key.

Free tier limits: 8 requests per hour · 15 requests per day

The API covers metrics including price, MVRV, NUPL, SOPR, Realized Price, Hashrate, exchange flows, address distributions, and many others. Full documentation is available at bitcoin-data.com/api/scalar.html.

Get your free API key at bitcoin-data.com/bguser/login.
Bitcoin Data API — Paid Plans
For projects that need higher request volumes, we offer paid subscription plans with significantly higher rate limits, access to all endpoints, and priority support.

Why upgrade?
• Higher hourly and daily request limits to support production workloads.
• Access to all API endpoints including block-level data.
• Suitable for dashboards, trading tools, research platforms, and automated pipelines.
• Payments supported via Lightning Network (Bitcoin) and other methods.

View full plan details and pricing at bitcoin-data.com/bguser/pricing.
Bitcoin Block API
In addition to our on-chain metrics API, we offer a Bitcoin Block API that provides block-level data directly from the Bitcoin blockchain.

Useful for applications that need raw block data, transaction details, fee analysis, mempool information, or custom on-chain calculations.

Full documentation and available endpoints: bitcoin-data.com/api-block/redoc.html.
Custom Chart Development (Highcharts & Plotly)
We are specialists in data visualization using Highcharts and Plotly. If you have data and need custom interactive charts — whether for a website, dashboard, report, or internal tool — we can build them for you.

• Time series, OHLC, area, scatter, radar, heatmaps, and more.
• Fully interactive: zoom, pan, crosshair, tooltip, export.
• Light and dark themes, responsive design, embeddable via iframe.
• We can work with any data source: CSV, JSON, REST API, or database.

Contact us at info@bgeometrics.com with your requirements.
Lightning Network Payment Integration
We offer consulting and development services to integrate Bitcoin payments via the Lightning Network into your project or website.

• Instant, low-fee Bitcoin payments without intermediaries.
• Recurring subscriptions, pay-per-use models, or one-time payments.
• Integration with existing backends (Python, Node.js, REST APIs).
• We use Lightning Network in production ourselves — see our own implementation at bitcoin-data.com/bguser/pricing.

Ideal for SaaS products, content platforms, or any project looking to accept Bitcoin natively.
Custom Bitcoin Analytics
We can build tailored analytics solutions combining Bitcoin on-chain data with any external data you provide — macro indicators, exchange data, social sentiment, or your own proprietary datasets.

• Custom metric definitions and calculation pipelines.
• Automated daily or real-time data updates.
• Interactive dashboards with multiple charts and views.
• Workspace layouts with multiple panels for simultaneous analysis.

Whether you need a single custom indicator or a full analytics suite, we can help you build exactly what you need.
Chart & Data Embedding
Integrating BGeometrics charts into your website or application is as simple as copying a single line of code. All charts are available as iframes and can be embedded in any HTML page, blog, or web app.

• Works with any CMS: WordPress, Ghost, Webflow, custom HTML.
• Responsive by default — adapts to any screen size.
• Automatic daily data updates — no maintenance required on your side.
• Light and dark mode variants available for every chart.

For API-based data integration, use our REST API to fetch metrics directly into your own frontend or backend.
Dashboards & Workspaces
We offer pre-built multi-chart dashboards and customizable workspaces where you can monitor multiple Bitcoin metrics simultaneously.

Available dashboards include: Short Term Trend, Demand, Miner Activity, Market Peaks, Market Bottoms, and custom workspace layouts with 4 or 8 simultaneous charts.

We can build custom dashboard layouts tailored to your specific workflow — for trading, research, portfolio monitoring, or media publication.
