Bitcoin Active Addresses

The number of unique Bitcoin addresses that sent or received transactions. Only how you activate addresses that made successful transactions are taken into account.

This is a metric that shows the activity in the Bitcoin network.

Currently there are daily movements that occur in exchanges and in layers 2 such as the Lightning Network that are not reflected in this graph.
