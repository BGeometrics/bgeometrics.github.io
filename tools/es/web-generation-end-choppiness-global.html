Bitcoin Choppiness (CHOP)

El Choppiness o índice de agitación (CHOP) es un indicador de análisis técnico que se utiliza para determinar si un mercado tiene una tendencia o cotiza lateralmente (sin tendencia definida).

Este indicador no muestra la dirección de la tendencia, sino más bien el nivel de "rotación" o "entrecortamiento" del mercado.

El índice tiene un rango de valores entre 0 y 100. Los valores cercanos a 100 indican que el mercado cotiza lateralmente (rango). Los valores cercanos a 0 sugieren que el mercado se encuentra en una tendencia fuerte (ya sea alcista o bajista).

Un índice de agitación alto sugiere que el mercado está "entrecortado", lo que significa que no hay una dirección clara y los precios fluctúan dentro de un rango. Un índice de agitación bajo indica que el mercado se encuentra en una tendencia fuerte, con un claro movimiento direccional.

Fue desarrollado por el comerciante E.W. Dreiss.
