Este es el sitio web provisional de bitcoin-data.com, un sitio web donde los datos de Bitcoin estarán disponibles de forma gratuita a través de una API REST; serán principalmente métricas On Chain de Bitcoin Blockchain.
Métricas On-Chain
Con la API de bitcoin gratuita puedes acceder fácilmente a las métricas BTC On-Chain más relevantes extraídas de la cadena de bloques de Bitcoin que ofrecen información sobre el precio actual de Bitcoin, tendencias del mercado, volúmenes de transacciones y mucho más.
