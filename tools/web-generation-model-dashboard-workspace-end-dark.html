Bitcoin Dashboard Workspace

Dashboard with four workspaces to be able to select different bitcoin graphics on the same page.

Initially, only the BTC price appears on the graph in the workbench, but different metrics can be selected and incorporated.

In the drop-down menus of each graph, different data and formula options can be chosen.
